# arm cpu

## Support Instructions

### Data Processing

* *Opcode* Rd, Rn, Src2
  * AND, SUB, ADD, ORR, ADC
* *Opcode* Rn, Src2
  * CMP, CMN, TST
* *Opcode* Rd, Rm, Rs/shamt5
  * LSL

### Memory

* *Opcode* Rd, [Rn]
  * STR, LDR

### Branch

* *Opcode* label
  * B
